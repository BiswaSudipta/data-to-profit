## 🚀 Getting Started

Run locally in just a few steps:

```bash
# Clone repo
git clone https://github.com/YourUsername/data-to-profit.git

# Enter project
cd data-to-profit

# Install dependencies
npm install

# Start dev server
npm run dev
```

👉 Open: `http://localhost:3000`
